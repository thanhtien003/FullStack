## Values Types - Reference Types

### 1. Values Types(Primitive data types) (Kiểu dữ liệu nguyên thủy)
>> Nhóm kiểu tham trị
    - String
    - Number
    - Boolean
    - BigInt
    - Symbol
    - Undefined
    - null

### 2. References Types(Non-primitive data types) (Kiểu dữ liệu phức tạp)
>> Nhóm kiểu tham chiếu
    - Object
    - Array
    - Function
